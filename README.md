# vue-cli

This project was cloned from [https://codesandbox.io/s/3y8nwrlr66](https://codesandbox.io/s/3y8nwrlr66) and is a simple Todo App build in Vue:

# Installation

- `git clone https://github.com/rkristelijn/vue-cli.git`
- `cd vue-cli`
- `npm i`
- `npm run serve`
